# YOLO-Based-Object-Detection
This study aims to evaluate the impact of Gaussian noise on the performance of YOLO-based object detection models and analyze the potential for improving model robustness through noise reduction combined with retraining. To achieve this goal, the study developed an experimental system using the PnPLO dataset for the human detection problem with 1,888 images. Five versions of the YOLOv8, YOLOv9, YOLOv10, YOLOv11, and YOLOv12 models were trained and evaluated on clean data, then tested on data with Gaussian noise at levels $\sigma = 0, 1, 5, 10, 20$, and $30$. Next, Gaussian noise reduction combined with model retraining was applied to evaluate the system's performance recovery. Experimental results show that Gaussian noise significantly degrades object detection performance. Specifically, as the noise level increased to $\sigma = 30$, the accuracy of YOLOv8 decreased from 0.7738 to 0.7566, while the F1-score of YOLOv11 decreased from 0.7753 to 0.7537, and the mAP50 and mAP50--95 indices also decreased significantly. After applying noise reduction and retraining, the performance of the models improved and became more stable; for example, YOLOv8 achieved an mAP50 = 0.8992 on clean data, and YOLOv9 maintained an accuracy of approximately 0.7696 at a noise level of $\sigma = 20$. The research results provide experimental evidence that combining Gaussian noise reduction and model retraining can significantly improve the stability of object detection systems. This method has practical value in deploying computer vision systems in real-world environments where image quality is often degraded by sensor noise or unstable lighting conditions. This study proposes a systematic experimental procedure to evaluate the impact of Gaussian noise on various YOLO instances and analyze performance resilience through a combined noise reduction and model retraining strategy. The results help clarify the relationship between input data quality, noise reduction techniques, and the robustness of modern object detection models.
